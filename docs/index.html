Fantastic-template

一款开箱即用的 Vue 项目模板

开始 →

全面支持 Vue 3

使用 Vue + Vite，默认集成 Vue-router & Pinia

全局 scss 资源引入

支持设置 全局 scss 资源，如：变量、函数、mixin

SVG 图标 & CSS 精灵图

SVG 图标自动载入，CSS 精灵图自动合成

Mock 数据

本地和生产环境均可使用 Mock 数据

文件压缩

支持 gzip / brotli 优化项目体积，提高加载速度

代码规范

结合 IDE 插件、ESlint 、stylelint 、Git 钩子，轻松实现团队代码规范
